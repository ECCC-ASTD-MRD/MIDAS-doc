link to source code

Dependency Diagrams:

No Direct Dependency

Description

MODULE clibInterfaces_mod (prefix=’’ category=’9. Global interfaces’)

Purpose

‘include’ all of the needed files containing the interfaces for the rmnlib clib library subroutines. This allows all other files in MIDAS to be of the ‘f90’ type (instead of ‘ftn90’), thus avoiding a pass through the pre-processor.
