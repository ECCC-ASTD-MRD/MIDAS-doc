rttovInterfaces_mod¶

link to source code

Dependency Diagrams:

No Direct Dependency

Description

MODULE rttovInterfaces_mod (prefix=’’ category=’9. Global interfaces’)

Purpose

To include all of the needed files containing the interfaces for the rttov library subroutines. This allows all other files in MIDAS to be of the f90 type (instead of ftn90), thus avoiding a pass through the pre-processor.
